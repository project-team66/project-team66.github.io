HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser. It defines the content and structure of web content. It is often assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

What is HTML used for?

Paragraphs

The HTML paragraph element is one of the most common elements and as you might have guessed it defines a paragraph.

Line Breaks

As with print media, a paragraph creates a line break below it to visually separate it from other paragraphs. This is used to emphasise a semantic separation of content. The same structure is used in a novel or a magazine.

Block Elements

Elements that create the spacing below themselves on a page are called block elements. Block elements appear vertically down the left-hand side of a page at least until they are styled by CSS. Headings

Paragraphs and headings work in concert to create the majority of the text content of a web page and its structure.
